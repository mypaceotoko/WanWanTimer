# 犬の鳴き声音声

`dog-bark.mp3.m4a` をここに置くと、タイマー終了時に本物の犬の鳴き声で「ワンワン！ワンワン！」とループ通知します。

- 現在の推奨パス: `assets/audio/dog-bark.mp3.m4a`
- iPhone Safari でも再生しやすいように、HTMLAudioElement で m4a 音声をループ再生します
- 将来的な差し替え用に、`assets/audio/dog-bark.mp3` と `assets/audio/dog-bark.m4a` も代替ファイルとして試します
- 著作権的に問題のないフリー素材、またはユーザー自身が用意・録音した音声を使用してください
- 外部URLには依存せず、GitHub Pagesで配信されるこのフォルダ配下に配置してください

このリポジトリでは、GitHub Pages で配信される音声ファイルとして `dog-bark.mp3.m4a` を使用します。
