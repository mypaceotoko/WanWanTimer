# ワンワンタイマー

かわいい犬型UIでカウントダウンし、時間が来たら犬の鳴き声で知らせるシンプルなタイマーです。

## 犬の鳴き声音声の配置方法

タイマー終了時は `assets/audio/dog-bark.mp3.m4a` を HTMLAudioElement で読み込み、iPhone Safari でも再生しやすい m4a 音声として、停止ボタンまたはリセットボタンが押されるまでループ再生します。

このリポジトリには、現在 `assets/audio/dog-bark.mp3.m4a` として犬の鳴き声音声を配置しています。差し替える場合は、以下の手順で権利的に問題のない音声ファイルを配置してください。

1. フリー素材サイトなどで、商用利用・再配布・Web公開の条件を確認した犬の鳴き声音声を用意する
2. ユーザー自身で録音した犬の鳴き声音声を用意する
3. ファイル名を `dog-bark.mp3.m4a` にして、`assets/audio/dog-bark.mp3.m4a` に配置する

将来的な差し替えに備えて、`assets/audio/dog-bark.mp3` と `assets/audio/dog-bark.m4a` も代替ファイルとして試します。`dog-bark.mp3.m4a`、`dog-bark.mp3`、`dog-bark.m4a` のいずれもない場合は、画面に案内を表示し、最低限のフォールバック音で通知します。

## 動作確認方法

1. `npm run start` でローカルサーバーを起動する
2. ブラウザで `http://localhost:4173/` を開く
3. 分を `0`、秒を `1` にして「セット」を押す
4. 「スタート」を押して、終了後に「ワンワン！時間です！」が表示されることを確認する
5. `assets/audio/dog-bark.mp3.m4a` を配置している場合は、犬の鳴き声がループ再生されることを確認する
6. 「大きな鳴き声停止ボタン」「鳴き声を止める」または「リセット」を押すと、鳴き声が完全に止まることを確認する

## 開発コマンド

```bash
npm run build
```

`npm run build` は `src/main.js` の構文チェックを実行します。
