# ワンワンタイマー

かわいい犬型UIでカウントダウンし、時間が来たら犬の鳴き声で知らせるシンプルなタイマーです。

## 犬の鳴き声音声の配置方法

タイマー終了時は `assets/audio/dog-bark.mp3` を HTMLAudioElement で読み込み、停止ボタンまたはリセットボタンが押されるまでループ再生します。

このリポジトリには著作権上の理由で実際の犬の鳴き声音声は同梱していません。以下のどちらかの方法で、権利的に問題のない音声ファイルを配置してください。

1. フリー素材サイトなどで、商用利用・再配布・Web公開の条件を確認した犬の鳴き声音声を用意する
2. ユーザー自身で録音した犬の鳴き声音声を用意する
3. ファイル名を `dog-bark.mp3` にして、`assets/audio/dog-bark.mp3` に配置する

`dog-bark.mp3` がない場合でも画面に案内を表示し、最低限のフォールバック音で通知します。本物の犬の鳴き声にしたい場合は、必ず上記パスに音声ファイルを置いてください。

## 動作確認方法

1. `npm run start` でローカルサーバーを起動する
2. ブラウザで `http://localhost:4173/` を開く
3. 分を `0`、秒を `1` にして「セット」を押す
4. 「スタート」を押して、終了後に「ワンワン！時間です！」が表示されることを確認する
5. `assets/audio/dog-bark.mp3` を配置している場合は、犬の鳴き声がループ再生されることを確認する
6. 「大きな鳴き声停止ボタン」「鳴き声を止める」または「リセット」を押すと、鳴き声が完全に止まることを確認する

## 開発コマンド

```bash
npm run build
```

`npm run build` は `src/main.js` の構文チェックを実行します。
